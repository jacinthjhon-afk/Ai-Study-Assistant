##  Tech Stack
- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend/DB**: Supabase (PostgreSQL, Auth, RLS)
- **Deployment**: Vercel
- **AI Integration**: Chatbot & Study Recommendations
- **Version Control**: GitHub (Public)
