#  AI Study Assistant Web App

A full-stack AI-powered study companion built with React, Supabase, and Vercel.

##  Tech Stack
- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend/DB**: Supabase (PostgreSQL, Auth, RLS)
- **Deployment**: Vercel
- **AI Integration**: Chatbot & Study Recommendations
- **Version Control**: GitHub (Public)
